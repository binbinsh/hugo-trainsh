# 欢迎来到 hugo-trainsh 演示站点

一个面向多语言内容场景的 Hugo 主题——**默认启用 NES/FC 复古像素风**。

{{< recent-posts >}}

## 可体验的特性

- **复古 NES 像素模式**——默认外观。深蓝背景、像素字体标题、8-bit 对话框边框、RPG 风格金色标签。点击手柄图标可切换到明亮和暗黑模式。
- 文章列表支持标签筛选和归档浏览
- 内置短代码：`toc`、`tags`、`recent-posts`
- 代码块支持语法高亮、复制按钮和自动换行
- Mermaid 图表与 KaTeX 数学公式
- PhotoSwipe 图片灯箱（支持本地图片尺寸信息）
- 可选文章点赞功能（示例配置已开启）
- 语言切换（EN / 简中 / 繁中 / 日本語）
