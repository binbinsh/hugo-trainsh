# 欢迎来到 hugo-trainsh 演示站点

一个面向多语言内容场景的 Hugo 主题，兼顾可读性与内容发现体验。

{{< recent-posts >}}

## 可体验的特性

- 文章列表支持标签筛选和归档浏览
- 内置短代码：`toc`、`tags`、`recent-posts`
- 代码块支持语法高亮、复制按钮和自动换行
- Mermaid 图表与 KaTeX 数学公式
- PhotoSwipe 图片灯箱（支持本地图片尺寸信息）
- 可选文章点赞功能（示例配置已开启）
- 主题色模式切换 + 语言切换
