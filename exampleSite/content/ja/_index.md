# hugo-trainsh デモへようこそ

多言語サイトに最適化された読みやすい Hugo テーマです。最新機能を一覧で確認できます。

{{< recent-posts >}}

## 特徴

- 投稿一覧でのタグ絞り込みとアーカイブ表示
- 組み込みショートコード: `toc`、`tags`、`recent-posts`
- シンタックスハイライト、コピー、ソフトラップ付きコードブロック
- Mermaid 図表と KaTeX 数式
- PhotoSwipe 画像ライトボックス（ページバンドル画像のサイズを自動取得）
- いいね API を使う任意機能（サンプル設定済み）
- テーマ切替と多言語切替
