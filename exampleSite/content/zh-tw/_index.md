# 歡迎來到 hugo-trainsh 示範站點

這是一個面向多語系內容的 Hugo 主題——**預設啟用 NES/FC 復古像素風格**。

{{< recent-posts >}}

## 特色

- **復古 NES 像素模式**——預設外觀。深藍背景、像素字體標題、8-bit 對話框邊框、RPG 風格金色標籤。點擊手把圖示可切換至明亮與暗黑模式。
- 文章列表支援標籤篩選與歸檔瀏覽
- 內建短代碼：`toc`、`tags`、`recent-posts`
- 程式碼區塊支援語法高亮、複製按鈕與自動換行
- Mermaid 圖表與 KaTeX 數學式
- PhotoSwipe 圖片燈箱（本機圖片可自動取得尺寸）
- 可選的文章按讚功能（示例已啟用）
- 語言切換（EN / 簡中 / 繁中 / 日本語）
